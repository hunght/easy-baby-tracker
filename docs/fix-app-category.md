## How to Fix App Category

### Step 1: Go to Store Settings

1. Open [Google Play Console](https://play.google.com/console/developers)
2. Select your app: **Easy Baby Tracker**
3. In the left sidebar, navigate to: **Store settings** > **App category**

### Step 2: Select Correct Category

You need to select **both** fields:

#### App or Game

- Select: **App** (not Game)

#### Category

- Select: **Parenting**

### Step 3: Save Changes

1. Click **Save** at the bottom
2. You should see a success message
3. The warning should disappear

## Common Issues

### Issue: Can't Save Category

**Solution**: Make sure you've filled in both fields:

- App or Game: **App**
- Category: **Parenting**

### Issue: Category Not Available

If "Parenting" is not available in the dropdown:

**Alternative Categories** that also work:

1. **Lifestyle** (most appropriate alternative)
2. **Health & Fitness** (also relevant)
3. **Medical** (if available)

### Issue: Changes Not Showing

**Solution**:

- Refresh the page
- Check if there's a pending review banner
- Wait a few minutes and check again
